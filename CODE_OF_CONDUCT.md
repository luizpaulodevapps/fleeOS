# Código de Conduta

Este projeto adota um código de conduta para promover um ambiente acolhedor e respeitoso.

## Padrões esperados

- Seja respeitoso com os outros.
- Aceite críticas de forma construtiva.
- Mantenha a comunicação profissional.
- Respeite a diversidade de opiniões e backgrounds.

## Comportamento não aceitável

- Ataques pessoais.
- Discriminação de qualquer tipo.
- Uso de linguagem ofensiva.
- Divulgação de dados sensíveis.

## Aplicação

Relate qualquer comportamento inadequado abrindo uma issue ou entrando em contato com o mantenedor do projeto.
